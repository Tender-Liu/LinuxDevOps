### 1.1 生活中的编程
想象你在教一个机器人做事情：

1. 机器人只能听懂非常明确的指令

* Python就是一种告诉计算机做事的语言
* 特点：简单、易学、容易读懂

## 四、变量入门

### 4.1 什么是变量
把变量想象成一个贴了标签的盒子：
* 盒子里可以放东西（数据）
* 标签就是变量名
* 随时可以改变盒子里的内容


### 4.2 使用变量
```bash
# 存储数字
age = 18
print(age)

# 存储文字
name = "小明"
print(name)

# 修改变量内容
age = 20
print(age)

```

### 4.3 变量的简单计算
```bash
# 数字计算
num1 = 10
num2 = 5
print(num1 + num2)   # 15
print(num1 - num2)   # 5

# 更新变量
score = 0
score = score + 10   # score现在是10
print(score)

```

```bash
num2 = input("请输入第二个数字：")

# 转换为数字
num1 = int(num1)
num2 = int(num2)

# 显示计算结果
print("计算结果：")
print(num1, "+", num2, "=", num1 + num2)
print(num1, "-", num2, "=", num1 - num2)
print(num1, "×", num2, "=", num1 * num2)
print(num1, "÷", num2, "=", num1 / num2)

```

## 附件--Python运算符一览表

### 算术运算符

| 运算符 | 描述 | 实例 | 结果 |
|--------|------|------|------|
| + | 加法 | 5 + 3 | 8 |
| - | 减法 | 5 - 3 | 2 |
| * | 乘法 | 5 * 3 | 15 |
| / | 除法 | 5 / 3 | 1.6666... |
| // | 整除（向下取整） | 5 // 3 | 1 |
| % | 取余 | 5 % 3 | 2 |
| ** | 幂运算 | 5 ** 3 | 125 |

### 比较运算符
| 运算符 | 描述 | 实例 | 结果 |
|--------|------|------|------|
| == | 等于 | 5 == 3 | False |
| != | 不等于 | 5 != 3 | True |
| > | 大于 | 5 > 3 | True |
| < | 小于 | 5 < 3 | False |
| >= | 大于等于 | 5 >= 3 | True |
| <= | 小于等于 | 5 <= 3 | False |

### 赋值运算符
| 运算符 | 描述 | 实例 | 等价于 |
|--------|------|------|--------|
| = | 简单赋值 | x = 5 | - |
| += | 加法赋值 | x += 3 | x = x + 3 |
| -= | 减法赋值 | x -= 3 | x = x - 3 |
| *= | 乘法赋值 | x *= 3 | x = x * 3 |
| /= | 除法赋值 | x /= 3 | x = x / 3 |
| %= | 取余赋值 | x %= 3 | x = x % 3 |
| **= | 幂赋值 | x **= 3 | x = x ** 3 |
| //= | 整除赋值 | x //= 3 | x = x // 3 |

### 逻辑运算符
| 运算符 | 描述 | 实例 | 结果 |
|--------|------|------|------|
| and | 与 | True and False | False |
| or | 或 | True or False | True |
| not | 非 | not True | False |

### 成员运算符
| 运算符 | 描述 | 实例 | 结果 |
|--------|------|------|------|
| in | 存在 | 'a' in 'abc' | True |
| not in | 不存在 | 'x' not in 'abc' | True |

### 身份运算符
| 运算符 | 描述 | 实例 | 结果 |
|--------|------|------|------|
| is | 是同一个对象 | a is b | True/False |
| is not | 不是同一个对象 | a is not b | True/False |

### 位运算符
| 运算符 | 描述 | 实例 | 结果 |
|--------|------|------|------|
| & | 按位与 | 5 & 3 | 1 |
| \| | 按位或 | 5 \| 3 | 7 |
| ^ | 按位异或 | 5 ^ 3 | 6 |
| ~ | 按位取反 | ~5 | -6 |
| << | 左移 | 5 << 1 | 10 |
| >> | 右移 | 5 >> 1 | 2 |

### 运算符优先级（从高到低）
| 优先级 | 运算符 |
|--------|--------|
| 1 | ** |
| 2 | ~, +, - (单目运算符) |
| 3 | *, /, //, % |
| 4 | +, - (双目运算符) |
| 5 | >>, << |
| 6 | & |
| 7 | ^, \| |
| 8 | <=, <, >, >= |
| 9 | ==, != |
| 10 | =, +=, -=, *=, /=, %=, //=, **= |
| 11 | is, is not |
| 12 | in, not in |
| 13 | not |
| 14 | and |
| 15 | or |
